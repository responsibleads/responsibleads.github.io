ResponsibleAds

Updated: 05/08/2019

This page provides an overview of the data behind the @ResponsibleAds Twitter account. This dataset is gathered from the Fox News West internet archive, which has been running since 2011. Each archive page includes a keyword summary for the show that often contains company names and products. I associated each of these products with a parent company and twitter account and tweet at them each day when they advertise.

Fox News, in it's current incarnation, amounts to a propaganda arm for a would-be authoritarian president. It's likely that well over half of Fox's revenue comes from advertising, so we can make a difference. Running propaganda should not be a profitable endeavor in our society, so join me in calling these companies out!

Sources of Bias

The web archive isn't a perfect source of data, so here are a few ways it could be biased:

  1. Closed captioning is missing for some ads. This could be random, or it could be that local and regional ads don't have closed captioning. I'm not too concerned about this, as I'm mainly interested in targeting the large players.
  2. Some companies might use their names more frequently in ads, which means they'll show up in the keywords more often.
  3. Some companies might heavily advertise on a single show and end up in the keywords as a result. Others might spread their ads across many shows so they won't be frequent enough to show up in the keywords.
  4. There might be sources of bias in the keyword summary process. I think their algorithm is just frequency based, but it might single out rare words instead.

The point of this analysis isn't to be perfect, but instead get a general sense of the big players. I won't open source the code for now because common coding patterns could be used to reveal my identity, but the data are available on GitHub and comments are welcome.
